# BankSystemServer

## 프로젝트 소개
* **Backend** : Spring Boot 3.2.5
* **Server** : Amazon EC2
* **Libraries** : Spring data JPA, Spring Security , JWT
* **Database** : Amazon RDS (MySQL)
* **Language** : Java
* **ETC** : Jenkins
